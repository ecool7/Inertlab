# Настройка отправки писем через Gmail

## Шаг 1: Создайте пароль приложения Gmail

1. Перейдите в [настройки аккаунта Google](https://myaccount.google.com/)
2. Включите двухфакторную аутентификацию (если еще не включена)
3. Перейдите в раздел [Пароли приложений](https://myaccount.google.com/apppasswords)
4. Выберите "Почта" и "Другое устройство"
5. Введите название: "Inertlab Site"
6. Скопируйте сгенерированный 16-значный пароль

## Шаг 2: Создайте файл .env

В корне проекта создайте файл `.env` со следующим содержимым:

```env
# Секретный ключ для Flask (сгенерируйте случайную строку)
SECRET_KEY=ваш-секретный-ключ-здесь

# Gmail адрес отправителя (ваш Gmail)
MAIL_USERNAME=ваш-email@gmail.com

# Пароль приложения Gmail (16-значный код из шага 1)
MAIL_PASSWORD=xxxx-xxxx-xxxx-xxxx

# Email куда будут приходить заявки (можно тот же Gmail или другой)
MAIL_TO=ваш-email@gmail.com

# SMTP настройки (обычно не нужно менять)
MAIL_SMTP_SERVER=smtp.gmail.com
MAIL_SMTP_PORT=587
```

## Шаг 3: Установите зависимости

```bash
pip install -r requirements.txt
```

## Шаг 4: Проверьте работу

Запустите сайт и отправьте тестовое сообщение через форму контактов.

## Важно!

- **НЕ** коммитьте файл `.env` в Git! Он уже должен быть в `.gitignore`
- Используйте **пароль приложения**, а не обычный пароль Gmail
- На PythonAnywhere установите переменные окружения в разделе "Web" → "Environment variables"
